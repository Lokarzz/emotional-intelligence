## _Emotional Intelligence_

## Tech Stack

- Kotlin
- Jetpack Compose
- MVVM (Model-View-ViewModel)
- Material 3 Design
- Dagger Hilt

- DataStore
- Glide

- git clone https://github.com/Lokarzz/emotional-intelligence.git
- open android studio and open the project
- run the app
